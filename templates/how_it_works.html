01
Data Collection
Raw Flight Dataset

The model was trained on a Kaggle dataset of ~300,000 domestic Indian flight records covering 6 airlines, 6 cities, and booking windows of 1 to 49 days before departure. Each row represents a real flight listing with its actual price.

SpiceJetAirAsiaVistara GO FIRSTIndiGoAir India ~300K rows
02
Preprocessing
Cleaning & Encoding

Categorical columns like airline, source city, stops, and time-of-day were converted into numbers using Label Encoding and One-Hot Encoding. Missing values were handled, and the duration column was standardized in hours.

Label EncodingOne-Hot Encoding Null handlingDuration normalization
03
Model Training
Random Forest Regressor

A Random Forest Regressor was selected after comparing it against a Decision Tree. Random Forest uses an ensemble of decision trees — each tree votes on a price, and the final prediction is the average. This reduces overfitting and handles non-linear relationships in pricing extremely well.

Decision Tree → 96% accuracy Random Forest → 98.6% accuracy 80/20 train-test split
04
Evaluation
Model Performance

The model was evaluated using R² Score (how much variance in price the model explains), Mean Absolute Error (average prediction error in ₹), and compared against a baseline Decision Tree to confirm the improvement.

R² Score: 0.986 MAE: ~1,200 ₹ Test size: 60,000 flights
05
Deployment
Flask + Render

The trained model is saved as a .pkl file using joblib and loaded by a Flask web server. When you submit the form, your inputs are preprocessed the same way as training data and passed to the model. The predicted price is returned instantly.
