Project overview

What Is SkyFare?

SkyFare is a flight price prediction web app that uses a trained Random Forest Regressor to estimate ticket prices for domestic Indian flights. You provide details like route, airline, class, stops, duration, and how many days before departure you're booking — and the model predicts the fare.

The project was developed as part of a Machine Learning & Artificial Intelligence internship (2026), demonstrating end-to-end ML development: data preprocessing, model training, evaluation, and deployment as a live web application.

The Dataset

The model was trained on a publicly available Kaggle dataset of Indian domestic flight prices. It contains approximately 300,000 records of real flight listings across 6 major airlines and 6 Indian cities, scraped from a flight booking platform.
